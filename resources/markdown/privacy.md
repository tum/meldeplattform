# Datenschutzerklärung

Diese Plattform wird von der Technischen Universität München betrieben.

## Anonymität

Meldungen können vollständig anonym erstellt werden. Es werden keine
IP-Adressen oder Browserkennungen langfristig gespeichert.

## Freiwillige Kontaktangabe

Wenn Sie bei Absenden einer Meldung eine E-Mail-Adresse hinterlassen,
wird diese ausschließlich verwendet, um Sie über Rückfragen und den
Status der Meldung zu informieren.

## Login für Administrator*innen

Für den Login verantwortlicher TUM-Mitarbeiter*innen wird das zentrale
Shibboleth-Identity-Management (SAML) der TUM verwendet.
